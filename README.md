# ⚡ Physics of Wires

An interactive JavaScript simulation demonstrating dynamic wire motion using physics principles and HTML5 Canvas rendering.

## 🚀 Live Demo
👉 https://abhinav20-pixel.github.io/Physics-of-Wires--AviNav/

## 📌 Features
- Smooth dynamic wire animation
- Physics-based curve movement
- HTML5 Canvas rendering
- Clean UI with minimal design
- Adjustable settings panel

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Canvas API

## 📂 Project Structure
physics-of-wires/
│── index.html
│── style.css
│── script.js

## 🎯 Purpose
This project was built to experiment with physics simulation concepts and interactive canvas-based rendering.
